## Overview
This project demonstrates hands-on experience in automating data gathering and performing **data wrangling** using the **Netflix Shows Dataset** from Kaggle.

## 🎯 **Objectives**

The project covers:

- Loading and exploring the dataset using **pandas**
- Performing data discovery: data types, missing values, duplicates
- Cleaning the dataset: handling blanks, formatting, and inconsistencies
- Structuring and transforming variables
- Validating data quality using logical checks
- Exporting the final cleaned dataset for future analysis
